# AI Prompting — Interactive Exercise Concepts

## For Future 90-Minute Session Expansion

These exercises are designed to be added when workshops expand from 30 to 90 minutes.

### Exercise 1: Lazy vs. RBTF Prompt (15 min)
- Groups of 2-3 students
- Each group picks a real research task one of them is working on
- Write a "lazy prompt" first — see what generic response they get
- Apply RBTF — add Role, Background, Task, Format
- Compare the two responses side-by-side
- Document the before/after

### Exercise 2: Iteration Chain (15 min)
- Start from the RBTF prompt from Exercise 1
- Iterate through 3 follow-up prompts using the refinement phrases
- Track how each iteration improves the response
- Identify which single change made the biggest difference

### Exercise 3: Format Matters (10 min)
- Same research question, but request different output formats:
  - Bullet points
  - Table
  - Prose paragraph
  - Step-by-step guide
- Compare: which format was most useful for the task?

### Exercise 4: Cross-Tool Comparison (15 min)
- Same prompt across different AI tools (ChatGPT, Claude, Gemini)
- Compare responses: depth, accuracy, style, usefulness
- Discuss: when might you prefer one tool over another?

### Exercise 5: Group Presentations (15 min)
- Each group shares (2-3 min each):
  1. The task — what research problem did you work on?
  2. Before — show lazy prompt and generic response
  3. After — show refined prompt and improved response
  4. Key insight — what single change made the biggest difference?
